\doxysubsection{Detailed Description}
Singleton class responsible for mutex management. 

The \mbox{\hyperlink{class_mutex_manager}{Mutex\+Manager}} class serves as a singleton that provides a QMutex instance for managing synchronization locks. This ensures a centralized control of mutex operations.
